# Changelog — April 5, 2026

## URL Input
- Hide the typed URL from the input field after clicking Go
- Added placeholder text: `type a url e.g. wikipedia.org/wiki/frog`
- Automatically prepend `https://` (falling back to `http://`) if the URL has no scheme
- Improved error text: shown below the URL input, styled to match the design system

## Feeling Lucky
- Added a "Feeling Lucky" button that picks a random URL from a preset list (`dial-a-site.json`)

## Past Walks Selector
- Each walk entry now shows a formatted date (e.g. Fri, May 2)
- Each page within a walk is on its own numbered line showing name + truncated URL
- Line numbers rendered in muted color
- List becomes scrollable when long, with balanced top/bottom margins

## Auto-Walk
- Off by default
- Resets to off whenever a new walk is started

## Frog Sprite
- Frog no longer pauses when off-canvas
- Increased pause duration when on-canvas
- Fixed occasional glitch when turning near canvas edges

## Path Connections
- Fixed visual inconsistency when the walk backtracks to a previously visited page

## Controls
- Cleaned up placement and grouping of UI controls

## Link Sampling
- Filter out boring/utility pages from sampled links: login, signup, sign-in, logout, register, forgot-password, reset-password, account, profile, settings, preferences, dashboard, privacy, terms, cart, checkout, payment, oauth, sso, callback, admin

## Misc
- Updated favicon for the Pond theme
- Added a `?` link to the docs site (docs.linkdrift.net)
- Fixed image expand: all page images (not just the active page) are now clickable to expand
